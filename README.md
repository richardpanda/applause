# Applause

[![Build Status](https://travis-ci.org/richardpanda/applause.svg?branch=master)](https://travis-ci.org/richardpanda/applause)

## Usage

Install dependencies
```
go get ./...
```

Run application
```
go run main.go --limit 25 --pages 10 --topic software-engineering
```
